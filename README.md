# Comparing Neural Networks for NHL Performance Forecasting

This is a course project for CSCI 4050U - Machine Learning

All code is included in worksheet.ipynb. The code is intended to be run in cell order. Training the models may take a few minutes.

The models used are:

- MLP
- RNN
- LSTM
- LSTM + MLP hybrid
